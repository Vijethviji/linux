	* Multiuser: Multiple users can access system resources simultaneously.
	* Multitasking: Can run multiple tasks simultaneously.

grep command "Global Regular Expression Print"

	* Grep command search for a particular string/keyword from a file and print lines matching a pattern.
	* It check line by line and print lines matching given pattern.
	* We can use grep anywhere like with files, searching for file, directories etc.
	
grep command syntax:
	grep [OPTION].. Pattern [File]..
	
use cases:
	To search kara name in file users.csv
	:- grep kara users.csv
	In this case it is case sensitive.
	
case1:
	To ignore the upper and lower case while searching:
	:- grep -i "keyword" file
	-i means ignore.
	
case2:
	To search everything except given pattern/keyword:
	:- grep -v "keyword" file
	-v invert match.
	
case3: 
	To print how manyh times (count)) given keyword present in file:
	:- grep -c "keyword" file
	-c count.

case4: 
	To search for exact match of given keyword in a file.
	:- grep -w "keyword" file
	Gives only specific word line.
	
case5:
	To print the line number of matches of given keyword in a file:
	:- grep -n "keyword" file
	
case6:
	To search a given keyword in multiple files:
	:- grep "keyword" file1 file2 ...
	grep -h "keyword" file1 file2 ..  (to suppress or hide the file name).
	
case7:
	To search multiple keywords in a file:
	:- grep -e "keyword1" -e "keyword2" file
	:- egrep "keyword1|keyword2|..." file (prefer this)..
	
case8:
	To search multiple keywords in multiple files.
	:- grep -e "keyword1" -e "keyword2" file1 file2
	
case9:
	To print file names which matches the given keyword:
	grep -l "keyword" file1 file2
	
case10:
	To get the keyword/pattern from a file and match with a another file:
	:- grep -f keyword.txt file
	
case11:
	To print the matching ;ine which start with given keyword:
	:- grep "^keyword" file (^ = start with)
	
case12:
	To pring the matching line which end with given keyword:
	grep "keyword$" file
	
case13:
	Suppose we have 100 files in a directory and we need to search a keyword in all the files:
	:- grep -R "keyword" directory/
	OR
	:- grep "keyword" directory/*
